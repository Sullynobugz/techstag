# TechStag - AI Agency Website

Navigate the AI Forest with Expert Guidance

## Overview

TechStag is a modern, professional MVP website for an AI agency and consulting company. Built with Next.js 14, TypeScript, and Tailwind CSS, it showcases AI services and provides a seamless user experience for potential clients.

## Features

- **Modern Design**: Clean, professional aesthetic with subtle AI-themed elements
- **Responsive**: Mobile-first design that works on all devices
- **Performance Optimized**: Fast loading with Next.js optimizations
- **SEO Ready**: Proper meta tags and semantic HTML
- **Interactive Elements**: Smooth animations with Framer Motion
- **Contact Form**: Functional contact form with validation
- **Conversion Focused**: Clear CTAs throughout the site

## Pages

- **Homepage**: Hero section, services overview, social proof, and CTA
- **Services**: Detailed AI service offerings
- **About**: Company mission, team, and why choose TechStag
- **Contact**: Contact form and business information

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- React Hooks

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Deployment

This project is optimized for Vercel deployment:

```bash
npm run build
```
